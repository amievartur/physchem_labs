# physchem_labs

## 📋 Обзор

Этот репозиторий содержит Jupyter Notebook с полным циклом обработки экспериментальных данных:

| Ноутбук | Метод | Физико-химическая задача |
|:---|:---|:---|
|`raman.ipynb`| Рамановское рассеяние | Определение энергии водородной связи |
| `dls.ipynb` | Динамическое рассеяние света (DLS) | Определение гидродинамического радиуса частиц по зависимости времени корреляции от волнового вектора |
|`nmr.ipynb`|Ядерно-магнитный резонанс|Определение времен релаксации|
| `gc.ipynb` | Газовая хроматография (GC) | Расчёт теплот адсорбции органических соединений на основе температурной зависимости удерживания |
|`spectras.ipynb`| Спектры водорода и йода | Вычисление энергии электронного перехода, диссоциации, константы Ридберга |
